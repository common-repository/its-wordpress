This Plugin helps to write WordPress , in it's correct spelling, by chageing the wrong one's to right ones.

== Description ==

"It's "WordPress"" is a simple WP plugin that will eliminate your mistakes from the post of the spelling WordPress or WP. If you write wordpress or Wordpress it will automatically make that WordPress in the post body and title. So you need no to worry for that anymore.

This will make auto correction of your spelling, so you will see "wordpress" in the editor , but the readers will see WordPress or WP in the page/post. 

== Installation ==



1. Upload `Its WordPress.zip` to the `/wp-content/plugins/` directory and extract the zipped file.
2. Activate the plugin through the 'Plugins' menu in WordPress
